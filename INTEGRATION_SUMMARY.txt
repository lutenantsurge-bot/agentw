SERIAL PROTOCOL (Arduino/ESP32)
═══════════════════════════════════════════════════════════════

Commands (newline-terminated):
  F<speed>   - Forward, speed 0-255
  B<speed>   - Backward
  L<speed>   - Turn left (differential)
  R<speed>   - Turn right
  S          - Stop
  X          - Emergency stop
  A<n>:<deg> - Servo arm n to degree 0-180

Defaults:
  Port: /dev/ttyUSB0
  Baud: 115200

═══════════════════════════════════════════════════════════════
IP WEBCAM CONFIGURATION
═══════════════════════════════════════════════════════════════

Android IP Webcam App requirements:
  - Install "IP Webcam" from Play Store
  - Start server on port 8080 (default)
  - Note phone IP address

Defaults:
  Host: 192.168.1.100
  Port: 8080

═══════════════════════════════════════════════════════════════
HOW TO BUILD AND RUN
═══════════════════════════════════════════════════════════════

1. Build with robot mode:
   cd ~/Desktop/llama-box-agentw
   ./build.sh -sbot

2. Run with custom settings:
   ./build/bin/llama-box-agent -sbot \
     -sbot-port /dev/ttyACM0 \
     -sbot-baud 115200 \
     -sbot-cam 192.168.1.50 \
     -sbot-cam-port 8080

3. Run without robot (basic):
   ./build/bin/llama-box-agent

TOOL CALLS FOR LLM

rc_control:
  {
    "name": "rc_control",
    "arguments": {
      "action": "forward",
      "speed": 160
    }
  }

Actions: forward, backward, left, right, stop, emergency_stop

webcam_capture:
  {
    "name": "webcam_capture",
    "arguments": {
      "save_as": "frame.jpg"
    }
  }

ARCHITECTURE FLOW
═══════════════════════════════════════════════════════════════

[LLM Response] 
     ↓
[Parse tool call / keywords]
     ↓
[Execute rc_control tool] 
     ↓

     ↓

     ↓
[L298N drives motors]

For IP Webcam:

[Agent needs vision]
     ↓
[webcam_capture tool called]
     ↓
[IPCam.grab_jpeg() → HTTP GET /shot.jpg]
     ↓
[JPEG bytes saved to file]
     ↓
[clip.cpp processes frame]
     ↓
[Visual embeddings → LLM]
